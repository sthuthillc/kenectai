Write HTML.

An open-source framework that turns HTML, CSS, and seekable animations into deterministic MP4 video — built for AI agents, automation, and CI.

$ npx @kenectai/cli init my-video
Get started Star on GitHub

▸Agent-native

Agents already write HTML. Twenty bundled skills teach them the full production loop — plan, compose, animate, add media, render.

▸Deterministic

Every frame is seeked in headless Chrome and encoded with FFmpeg. Same input, same bytes — safe for CI and regression tests.

▸Cloud or local

Render free on your machine, or push to KENECT AI Cloud with kenectai cloud render — zero infrastructure to manage.

<!-- a composition is just an HTML file -->
<div data-composition-id="teaser" data-start="0" data-width="1920" data-height="1080">
  <h1 class="clip" data-start="1" data-duration="4">Ship it.</h1>
</div>

$ kenectai render   # → teaser.mp4, frame-for-frame reproducible
