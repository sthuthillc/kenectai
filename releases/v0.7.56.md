# KENECT AI v0.7.56

Released on 2026-07-14.

Fixes a real bug affecting every installed copy: the update-check pinged the upstream `hyperframes` npm package instead of `@kenectai/cli`, so the CLI nagged users to "upgrade" against a version number that belonged to a different package. Also removes the last of the HyperFrames branding from CLI output (banner, User-Agent header, error messages, deployment defaults) and migrates catalog doc media off HeyGen's asset bucket onto our own.
