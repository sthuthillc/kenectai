# KENECT AI v0.7.57

Released on 2026-07-14.

Fixes `npm install @kenectai/cli` (and `npx`) failing entirely — not just background-removal — whenever the optional `onnxruntime-node` native binary can't be fetched, which happens on any network-restricted install (corporate proxy, sandboxed CI, sandboxed agent environments). It's now a proper optional dependency, matching how the code already treated it at runtime.
